# Accordion

Accordion component.

#### Example

<Canvas of={Accordion.Default} />

## Arguments

- **allowMultipleOpen** _`boolean`_ - Indicator for whether to allow multiple open items. _(default: false)_ _(optional)_
- **title** _`string`_ - The title to display for the accordion. _(optional)_
- **items** _`Item[]`_ - Each item in the accordion. _(default: \[\])_ _(optional)_

### Item

- **title** _`string`_ - The title to be displayed to for the accordion item.
- **children** _`ReactNode`_ - The components to be displayed when opening the accordion item.

## Usage

```ts
import {Accordion} from "@nick8green/components";

const Example = () => {
  return (
      <Accordion
        items={[
        {
            title: "Item 1",
            children: <div>Content for Item 1</div>,
        },
        {
            title: "Item 2",
            children: <div>Content for Item 2</div>,
        },
        {
            title: "Item 3",
            children: <div>Content for Item 3</div>,
        },
        {
            title: "Item 4",
            children: <div>Content for Item 4</div>,
        },
        {
            title: "Item 5",
            children: <div>Content for Item 5</div>,
        },
        ]}
        title="Accordion Title"
      />
  );
};

export default Example;
```

## Supported Styling

| Usage                                                                             | Variable                  | Alternatives (in priority order) | Fallback      |
|-----------------------------------------------------------------------------------|---------------------------|----------------------------------|---------------|
| The text colour for the title and the accordion title background when active/open | --colour-primary          | --colour-background-primary      | darkslategrey |
| The default background colour for the item title                                  | --colour-secondary        | --colour-background-secondary    | darkgreen     |
| The border colour separating options                                              | --accordion-border-colour | --colour-border                  | black         |
| The default text colour for the item title                                        | --colour-primary          | --colour-background-primary      | white         |
| The accordion title text colour when active/open                                  | --colour-secondary        | --colour-background-secondary    | white         |
