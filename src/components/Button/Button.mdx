# Button

Button component.

#### Example

<Canvas of={Button.Default} />

## Arguments

- **disabled** _`boolean`_ - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with. _(optional)_
- **label** _`string`_ - A string that represents the text content of the button. _(optional)_
- **onClick** _`() => void`_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element. _(optional)_
- **type** _`ButtonType`_ - The type of button to be used. Default to 'button'. _(optional)_

## Usage

```ts
import {Button} from "@nick8green/components";

const Example = () => {
  return (
      <Button
        label='Button'
        onClick={()=> console.log("Clicked")}
      />
  );
};

export default Example;
```
