# Button

Button component with different props.

#### Example

<Canvas of={Button.Default} />

## Usage

```ts
import {Button} from "@nick8green/components";

const Example = () => {
  return (
      <Button
        label='Button'
        onClick={()=> console.log("Clicked")}
      />
  );
};

export default Example;
```

#### Arguments

- **text** _`string`_ - A string that represents the text content of the button.
- **type** _`ButtonType`_ - The type of button to be used. Default to 'button'.
- **disabled** _`boolean`_ - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **onClick** _`() => void`_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
