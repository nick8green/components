# Button Group

Button Group component for holding buttons in a group on the front end.

#### Example

<Canvas of={ButtonGroup.Default} />

## Arguments

None

## Usage

```ts
import {Button, ButtonGroup} from '@nick8green/components';

const Example = () => {
  return (
      <ButtonGroup>
        <Button text='Done!' />
        <Button text='Reset' type={ButtonType.RESET} />
      </ButtonGroup>
  );
};

export default Example;
```
