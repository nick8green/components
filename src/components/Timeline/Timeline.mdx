# Timeline

This is a timeline component which renders a list of events in chronological order. It can be in ascending or descending order. The list of items is required and that list must include the date and the content to display. The content for each even can be just a string or it's own react element. The list is a vertical list at this time with each item alternating sides.

#### Example

<Canvas of={Timeline.Default} />

## Arguments

- **dateFormat** _`string`_ - The format that any dates should be displayed in. _(default: "DD/MM/YYYY")_ _(optional)_
- **dateLocation** _`"default" | "alternate"`_ - Whether the dates should remain with each event or be on the alternate side. _(default: "default")_ _(optional)_
- **display** _`"vertical"`_ - The orientation of the timeline. At this point, it can only be vertical. _(default: "vertical")_ _(optional)_
- **order** _`"asc" | "desc"`_ - The order that the events should be listed. _(default: "asc")_ _(optional)_
- **items** _`Item[]`_ - The events to be listed on the timeline. _(default: [])_

### Timeline Item Props

- **content** _`string | ReactNode`_ - The content to be displayed for the event. It can be a string which will be output in markdown or it can be a react component itself.
- **date** _`Date`_ - The date of the event/item.
- **icon** _`ReactNode`_ - An icon or element to be displayed on the timeline itself.
- **link** _`string`_ - A link that can take the user to a new page to get greater detail.
- **tag** _`string`_ - A tag which can be displayed on each event.
- **title** _`string`_ - The title of the event if there is to be one.
