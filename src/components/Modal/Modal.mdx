# Modal

Modal component. There are two types of modals, `LOADER` and an `INPUT` modal. If it is an input modal, there must be a close callback passed in for the actions that should occur when the modal is closed.

#### Example

<Canvas of={Modal.Default} />

## Arguments

- **close** _`() => void`_ - Any action that should happen as the result of clicking to close the modal. _(optional)_
- **type** _`ModalType`_ - The type of modal to display. _(optional)_
- **visible** _`boolean`_ - Whether the model should be visible.

## Usage

```ts
import {Modal} from "@nick8green/components";

const Example = () => {
  return (
    <Modal visible={true}>
      <div>
        <h1>This is a modal display</h1>
        <p>This is where we display overlay messages</p>
      </div>
    </Modal>
  );
};

export default Example;
```
