# Endorsement

Endorsement component for a website.

#### Example

<Canvas of={Endorsement.Default} />

## Arguments

- **comment** _`string`_ - The comment made to form the endorsement.
- **date** _`Date`_ - The date of the endorsement. _(optional)_
- **location** _`string`_ - The location of the user/individual/business making the comment. _(optional)_
- **name** _`string`_ - The name of the user/individual/business making the comment. _(optional)_

## Usage

```ts
import { Endorsement } from "@nick8green/components";

const Example = () => {
  return (
      <Endorsement
        comment="This is an endorsment"
        date={new Date(2023, 01, 01)}
        location="Leeds, UK"
        name="N8G"
      />
  );
};

export default Example;
```
