## Arguments

- **displayHandle** _`boolean`_ - Indicator as to whether to display the user's handle. _(optional)_
- **handle** _`string`_ - The user's social media handle.
- **platform** _`Platform`_ - The social media platform.
- **url** _`string`_ - The URL to access the user's social profile.

## Usage

```ts
import { SocialMediaLinks, SocialMediaPlatform } from "@nick8green/components";

const Example = () => {
  return (
      <SocialMediaLinks
        handle: "My Facebook",
        platform: Platform.Facebook,
        url: "https://www.facebook.com/",
      />
  );
};

export default Example;
```
