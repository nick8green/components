# Icon

Icon rendering component.

#### Example

<Canvas of={Icon.Default} />

## Arguments

- **pack** _`brand | solid`_ - The pack to pull the icon from.
- **name** _`string`_ - The name of the icon to be displayed from the pack. This is the key in the icon directory (see below).
- **size** _`"xs" | "sm" | "lg" | "1x" | "2x" | "3x" | "4x" | "5x"`_ - The size of the icon. _(default: 1x)_ _(optional)_
- **className** _`string`_ - Any class names to be applied to the icon. _(optional)_
- **title** _`string`_ - The title of the icon for accessibility. _(optional)_

## Usage

```ts
import { Icon } from "@nick8green/components";

const Example = () => {
  return (
    <Icon pack="solid" name="home" />
  );
};

export default Example;
```
